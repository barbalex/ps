# Exigences techniques

arten-fördern.app fonctionne dans le navigateur. Lequel importe peu en général, mais il devrait toujours être à jour.

arten-fördern.app fonctionne sur tous les systèmes d'exploitation.

En tant qu'[application web progressive](https://web.dev/explore/progressive-web-apps), vous pouvez également installer arten-fördern.app (la procédure dépend du navigateur). Elle s'exécute ensuite dans sa propre fenêtre, sans barre d'adresse, et peut être lancée depuis un raccourci — comme n'importe quelle autre application.

Vous n'avez pas à vous soucier des mises à jour ; elles s'effectuent automatiquement.
