# Geschichte

_Zuletzt aktualisiert am 3.5.2026_

---

Jedes Mal, wenn du einen Datensatz änderst, wird die alte Version auf dem Server aufbewahrt.

Wieso? Weil beim offline arbeiten Konflikte entstehen können. Mehr dazu [hier](/docs/offline).

Du kannst also jederzeit nachschauen, wer wann was geändert hat und die aktuelle Version wenn nötig gleich anpassen.

Da alte Versionen nur auf dem Server aufbewahrt werden, funktioniert das nur online.
