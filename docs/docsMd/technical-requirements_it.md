# Requisiti tecnici

_Ultimo aggiornamento: 7 maggio 2026_

---

arten-fördern.app funziona nel browser. Quale browser si utilizzi è generalmente irrilevante, ma dovrebbe essere sempre aggiornato.

arten-fördern.app funziona su qualsiasi sistema operativo.

Come [Progressive Web App](https://web.dev/explore/progressive-web-apps), è possibile installare arten-fördern.app. Le modalità esatte dipendono dal browser. Dopodiché viene eseguita in una propria finestra, senza barra degli indirizzi.

È quindi possibile avviare arten-fördern.app di norma anche da un collegamento. A seconda del sistema operativo, si comporta generalmente come qualsiasi app ottenuta dall'app store o installata direttamente.

Non è necessario occuparsi degli aggiornamenti; avvengono automaticamente.
