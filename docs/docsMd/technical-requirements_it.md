# Requisiti tecnici

Ultimo aggiornamento: 17 aprile 2026

arten-fördern.app funziona nel browser. Quale browser si utilizzi è generalmente irrilevante, ma dovrebbe essere sempre aggiornato.

arten-fördern.app funziona su qualsiasi sistema operativo.

Come [Progressive Web App](https://web.dev/explore/progressive-web-apps), è possibile installare arten-fördern.app (le modalità dipendono dal browser). Dopodiché viene eseguita in una propria finestra, senza barra degli indirizzi, e può essere avviata da un collegamento — come qualsiasi altra app.

Non è necessario occuparsi degli aggiornamenti; avvengono automaticamente.
