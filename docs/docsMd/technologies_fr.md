# Technologies utilisées

- [Akamai](https://www.linode.com) pour les serveurs physiques
- [Docker](<https://fr.wikipedia.org/wiki/Docker_(logiciel)>) pour faire fonctionner l'app sur le serveur
- [PostgreSQL](https://www.postgresql.org) comme base de données sur le serveur
- [Better Auth](https://better-auth.com) pour l'authentification
- [electric](https://electric-sql.com/sync) pour la synchronisation
- [PGlite](https://pglite.dev) comme base de données dans l'app
- [Jotai](https://jotai.org) pour gérer l'état de l'app
- [React](https://react.dev) pour l'interface utilisateur
- [Vite](https://vite.dev) comme environnement de développement
- Avec l'aimable assistance de [Claude](https://claude.com/product/claude-code)
