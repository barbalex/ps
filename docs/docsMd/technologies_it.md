# Tecnologie utilizzate

Ultimo aggiornamento: 17 aprile 2026

- [Akamai](https://www.linode.com) per i server fisici
- [Docker](<https://it.wikipedia.org/wiki/Docker_(software)>) per eseguire l'app sul server
- [PostgreSQL](https://www.postgresql.org) come database sul server
- [Better Auth](https://better-auth.com) per l'autenticazione
- [electric](https://electric-sql.com/sync) per la sincronizzazione
- [PGlite](https://pglite.dev) come database nell'app
- [Jotai](https://jotai.org) per gestire lo stato dell'app
- [React](https://react.dev) per l'interfaccia utente
- [Vite](https://vite.dev) come ambiente di sviluppo
- Con il gentile supporto di [Claude](https://claude.com/product/claude-code)
