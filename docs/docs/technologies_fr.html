Technologies utilisées

Dernière mise à jour: 17 avril 2026

  • Akamai pour les serveurs physiques
  • Docker pour faire fonctionner l'app sur le serveur
  • PostgreSQL comme base de données sur le serveur
  • Better Auth pour l'authentification
  • Electric pour la synchronisation
  • PGlite comme base de données dans l'app
  • Jotai pour gérer l'état de l'app
  • React pour l'interface utilisateur
  • Vite comme environnement de développement
